# Vanilla CSS Project

A simple website for a fictional travel agency.

Uses gulp as the main workflow with supporting packages.

## Technologies

### Stack
Project is created with: 
* HTML
* CSS/SCSS
* JS
* JQuery

### Packages
Project uses the following packages: 
* gulp
* esbuild

## Launch
The entry point is in app/index.html
